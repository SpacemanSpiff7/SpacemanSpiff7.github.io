# Archived Design Direction

This file is retained as historical reference only.

It captured an early dark-theme concept during initial ideation. It does not describe the shipped app or current repo state. The live app now uses:

- A warm light theme, not the dark teal palette from the original concept
- Three functional concentric rings, not a single-ring month scrubber
- A timeline-only primary view, not a bars/grid toggle
- Plain text labels, not emoji-based UI elements
- Shared visual treatment for local and import bars, with labels distinguishing source
- Fraunces + DM Sans as the implemented typography pair

Still-useful references from the original concept:

- Stoumann-inspired geometric background composition
- Matt Juggins-inspired rotating ring mechanics
- Staggered reveal as a general animation reference

If this file is reused in the future, reconcile every implementation detail with the current app before copying it into any active planning document.
