Quick Start Guide

Agile This is a lightweight task board that runs entirely in your browser. No account needed -- your data stays in localStorage.

Board Basics

Each project can have multiple boards. Switch between them using the sidebar tabs. Double-click a tab to rename it, or drag tabs to reorder.

Columns & Tickets

Drag tickets between columns to update their status. Use the quick-add input at the top of any column to create tickets fast. Each ticket can have a priority, labels, and a description.

Settings & Customization

Click the gear icon on any board tab to open its settings. You can customize columns, priorities, label presets, and the board accent color. Use the "Defaults" button in the header to set project-wide defaults for new boards.

Board colors

Import / Export

Export your project as a JSON file from the header. When importing, you have three options: add tickets to the active board, add as a new board in the current project, or create an entirely new project.

Multi-Project

Use the "New" dropdown to create additional projects. Each project gets its own URL (#project/0, #project/1, etc.) and is stored independently in localStorage. You can also duplicate an existing project or clear it from the same dropdown.
